# Ultimate React + FastAPI + MongoDB Boilerplate

## Features
- React (Vite) Frontend with Axios & Router
- FastAPI Backend with JWT Authentication
- MongoDB Connection (Motor)
- Login, Signup, Forgot Password, and Protected Routes included

## How to Run

### 1. Start Backend
cd server
venv\Scripts\activate  # Windows
uvicorn main:app --reload

### 2. Start Frontend
cd client
npm run dev
